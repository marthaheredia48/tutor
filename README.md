# 🤖 Tutor

Tutor is an offline AI tutoring application built with **SwiftUI** that runs the **Qwen 3 4B (4-bit quantized)** Large Language Model entirely on-device using **Apple MLX**. The application delivers fast, private, and internet-independent AI tutoring without relying on external APIs or cloud services.

## Features

- 🤖 Powered by **Qwen 3 4B (4-bit quantized)** running locally with **Apple MLX**
- 🔒 Fully offline inference with no external AI APIs
- 📎 Upload PDF, Markdown, TXT, and image files as contextual knowledge
- 🔍 Automatic OCR for scanned PDFs and images using Apple Vision
- ⚡ Real-time streaming token generation
- 🎓 Customizable AI tutor through configurable system prompts
- 💬 Supports reasoning output for compatible Qwen models

## Tech Stack

- SwiftUI
- Apple MLX
- MLX LLM
- Qwen 3 4B (4-bit)
- Hugging Face
- Vision Framework
- PDFKit
- Combine

## Requirements

- macOS (Apple Silicon recommended)
- Xcode 16+
- Swift 6
